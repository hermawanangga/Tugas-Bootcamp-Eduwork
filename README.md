### Tugas Sesi 4 - Responsive Web Portfolio

- HTML
  Menambahkan struktur halaman Portfolio Website yang responsif.
  Menambahkan Navbar dengan menu navigasi dan tombol Download CV.
  Menambahkan Hero Section sebagai landing page.
  Menambahkan About Me beserta informasi dan skill.
  Menambahkan Services Section.
  Menambahkan Project/Portfolio Section.
  Menambahkan filter kategori project (All, Web Design, UI/UX, dll).
  Menambahkan Testimonials Section menggunakan Swiper.js.
  Menambahkan Contact Section.
  Menambahkan Footer.
  Menambahkan Modal Contact dan Download CV.
  Menambahkan asset gambar dan icon pendukung.

- CSS
  Menggunakan CSS Custom Properties (CSS Variables) agar pengelolaan style lebih mudah.
  Menerapkan desain modern dengan warna, typography, spacing, shadow, dan border radius yang konsisten.
  Membuat tampilan responsive untuk desktop, tablet, dan mobile.
  Menambahkan animasi hover pada tombol, card, navbar, dan elemen interaktif.
  Menyesuaikan layout setiap section agar lebih rapi dan konsisten.

- JavaScript
  Mengimplementasikan fitur filter project berdasarkan kategori.
  Memperbaiki logika filter sehingga hanya card dengan kategori yang dipilih yang ditampilkan.
  Menambahkan slider testimonial menggunakan Swiper.js.
  Mengatur autoplay, pagination, dan responsive breakpoint pada slider testimonial.
  Memperbaiki bug pada fitur portfolio filter sehingga setiap tombol kategori hanya menampilkan project yang sesuai.

🎯 Hasil
Portfolio Website berhasil dibuat dengan tampilan modern, responsive, serta memiliki fitur interaktif menggunakan Bootstrap, CSS, dan JavaScript.
